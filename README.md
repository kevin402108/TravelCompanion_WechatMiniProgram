# 旅伴奇遇工坊

- 基于此项目代码实现的小程序主要包含以下功能：生成旅游路线、生成个性化旅游方案、在线组队（匹配旅伴、发起组队、查看及发布帖子）
- 采用 **npm** (version:9.8.1, 对应node版本：18.18.0) 和 **poetry** (version:2.1.4) 分别对前后端代码进行依赖管理，本项目主要包含以下文件/文件夹：
  - **frontEnd**：小程序前端代码
  - **backend**：小程序后端代码
  - **.gitignore**：项目整体忽略文件
  - **README.md**：项目说明文件
- WARNING: 由于采用**git**进行整个项目的版本管理，部分文件必须忽略（详见 **.gitignore**）,缺失的文件或文件夹需自行补全后方可使用！！!
